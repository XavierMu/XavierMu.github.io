Surfboard 使用教程

Surfboard 是一款专为 Android 安卓系统设计的高速科学上网代理工具，其名字直译为“冲浪板”，寓意帮助用户轻松冲破网络封锁。Surfboard 支持导入 Surge 的标准配置文件，用户可以直接使用 Surge 配置，简化设置过程。

Surfboard 通过 VpnService API 拦截系统流量，进行转发、修改或屏蔽操作，目前支持多种常见协议，包括 HTTP、HTTPS、SOCKS5、SOCKS5 over TLS、SS/SS-OBFS、Vmess、Trojan，但官方明确表示不支持 SSR 协议。

下载和安装

点击这里下载 Surfboard 安装包。

提示：Surfboard 是手机客户端，需要配合代理节点使用。

推荐机场

  • 魔界.net
  • WindyCloud

获取订阅地址

进入你选择的机场服务，购买适合的套餐后，复制订阅地址，或者在机场后台直接导入 Surfboard。

配置代理

  1. 如果你需要手动导入配置文件，打开 Surfboard，在配置页面点击右上角的加号，粘贴订阅地址，并导入代理配置。

  2. 配置完成后，进入仪表盘页面，选择合适的代理模式：

    • 规则模式（Rule）：根据配置文件的规则进行流量分流，仅需要代理的流量会经过代理服务器。推荐使用。
    • 直连模式（Direct）：所有流量直接连接到目的地，无需经过代理。
    • 全局模式（Global）：所有流量都将通过代理服务器。

  3. 在代理界面中，根据你访问网站的服务器位置或延迟来选择最佳代理节点，以确保流畅的网络体验。

开启代理

在仪表盘界面，点击右下角的按钮启动系统代理，代理生效后即可开始科学上网。

开始使用

成功启动代理后，你可以开始访问被封锁的网站和服务。建议首次使用时在浏览器的隐私模式下打开Google或YouTube进行测试，避免浏览器缓存或插件影响代理的正常工作。

通过这几步，你可以快速配置并使用 Surfboard 来进行网络加速和解锁被墙的网站。
