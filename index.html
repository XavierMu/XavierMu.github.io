Stable Diffusion是2022年发布的深度学习文本到图像生成模型。它主要用于根据文本的描述产生详细图像，尽管它也可以应用于其他任务，如内补绘制、外补绘制，以及在提示词指导下产生图生图的转变。

Stable Diffusion的源代码和模型权重已分别公开发布在GitHub和Hugging Face，可以在大多数配备有适度GPU的电脑硬件上运行。而以前的专有文生图模型（如DALL-E和Midjourney）只能通过云计算服务访问。

Surfboard 使用教程

Surfboard 是一款专为 Android 安卓系统设计的高速科学上网代理工具，其名字直译为“冲浪板”，寓意帮助用户轻松冲破网络封锁。Surfboard 支持导入 Surge 的标准配置文件，用户可以直接使用 Surge 配置，简化设置过程。

Surfboard 通过 VpnService API 拦截系统流量，进行转发、修改或屏蔽操作，目前支持多种常见协议，包括 HTTP、HTTPS、SOCKS5、SOCKS5 over TLS、SS/SS-OBFS、Vmess、Trojan，但官方明确表示不支持 SSR 协议。

LSPosed 安装教程

Xposed 框架 是一款能够在 不修改 APK 文件 的情况下影响程序运行（即修改系统行为）的强大框架服务。基于此框架，可以开发出众多 功能强大 的模块，并且在功能不冲突的前提下，这些模块可以同时运作。

LSPosed 是 Edxposed 的一个新分支，LSPosed 不仅与 Edxposed 相互兼容，而且相比 Edxposed 更加 简洁、易用和高效，成为了当前备受争议的 Edxposed 的绝佳 替代方案。

与 Xposed 类似，LSPosed 可以在 不实际修改任何应用和系统组件 的情况下达到修改目的，从而实现强大的功能。

2024年1月8日晚，LSPosed 项目团队宣布停止 LSPosed 模块管理软件项目的更新及维护工作。

ROOT 环境隐藏

Shamiko是一个与Magisk一起使用的模块，它可以在开启Zygisk的情况下隐藏root权限。这对于需要在root过的设备上使用那些通常不允许root权限的应用（如银行、金融和游戏类应用）非常有用。

Shamiko模块可以隐藏Magisk、Zygisk本身、二进制文件”su”，甚至在某些设备上隐藏”Bootloader未锁定”的状态。安装Shamiko后，默认是黑名单模式，这意味着你需要使用Magisk的排除列表来对特定应用隐藏root。
